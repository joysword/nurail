This project has developed an web-based interactive map tool that allows users to zoom in to an area to view environmental, demographic and economic data. The tool generate maps of the selected geographic area and environmental, demographic and economic data. Users can also selectively add geographic features to the map, such as public transit facilities, railroads, demographic data, water bodies, and vacant land.

What is the interactive map tool?

It is a Web-based interactive mapping tool for viewing and querying environmental and community information by user specified location (e.g., a street address, a transit stop, a neighbourhood name), or by rail planning goals (including environmental sustainability, equity, safety, and livable communities).

What is a layer?

A "layer" used in the Geographic Information System (GIS) community refers to a data source that defines how spatial data are visually represented on the map. Examples include: railroad lines, wetlands polygons, and other points of interest.

How to view the data layer(s)?

First click on the box next to the sub-folder name, then click on the specific data layer(s). For example, to view the data layer of “Grade Crossing,” check the box next to “Rail Infrastructure” before checking the box next to “Grade Crossing”.

How can I find a layer of interest?

Should you have difficulties in locating specific data layers hidden in the sub-folders, you can search all the relevant data layers that are available on this web site by entering a keyword (such as “wetlands”) in the “Find Layer Path” panel to the right of the map.

What functionalities does it provide?

Manipulations of the map include zooming in, zooming out, selecting layer, overlaying multiple layers, clearing layer selection, showing property information about a particular feature (e.g. a rail road segment, a grade crossing point, a transit facility). The manual states the usage of the map in details.

Which browser should I use?

We recommend Firefox or IE for map viewing. Depending on the size of specific data layer(s) and internet connection speed, you may experience slow responses at times.
